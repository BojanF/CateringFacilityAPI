API service documentation

  1. Return all active facilities registered:

    /api/[key]/active-facilities

  2. Details for given facility

    /api/[key]/facility-details/[facilityId]

  3. Contacts for given facility location for one facility:

    /api/[key]/facility-location-contacts/[locationId]

  4. All courses from one type for given facility

    /api/[key]/facility/[facilityId]/courses/[courseType]

  5. All courses from one type in all facilities:

    /api/[key]/courses/[courseType]

  6. All beverages from one type for given facility

    /api/[key]/facility/[facilityId]/beverages/[beverageType]

  7. All beverages from one type in all facilities:

    /api/[key]/beverages/[beverageType]

  8. All beverages types for given facility:

    /api/[key]/facility/[facilityId]/beverage-types-in-menu

  9. All courses types for given facility:

    /api/[key]/facility/[facilityId]/course-types-in-menu

  10. Details for given menu item, course or beverage:

    /api/[key]/detailed-menu-item/[menuItemId]
